# Optional Legal Pack

Bu dizin hukuk modülünü içerir.
Varsayılan olarak DEVRE DIŞI.

Etkinleştirmek için: SERIAI_ENABLE_LEGAL_PACK=true

Eski ALI projesinden taşınabilecek dosyalar:
- knowledge/tck/ → buraya kopyalanabilir
- knowledge/cmk/ → buraya kopyalanabilir
- knowledge/savunma_teknikleri/ → buraya kopyalanabilir

Tool'lar:
- legal_search
- legal_doc_forge
- sentencing_calculator

Bu dosyalar SERIAI'nin varsayılan akışını ETKİLEMEZ.
Sadece enable_legal_pack=true ise yüklenir.
